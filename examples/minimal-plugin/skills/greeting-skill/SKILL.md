# Greeting Skill

Use this skill when the user wants to create greetings or introductions.

## Guidelines

1. Be warm and friendly in tone
2. Consider the audience (formal vs informal)
3. Keep greetings concise but welcoming
4. Include relevant context when introducing

## Examples

### Informal Greeting
"Hey there! Welcome to our project. We're excited to have you here!"

### Formal Greeting
"Welcome to [Project Name]. This documentation will guide you through the setup process."

## When to Use

- Writing README introductions
- Creating welcome messages
- Drafting team introductions
- Composing greeting emails
