# Hello Command

A simple greeting command.

## Usage

```
/minimal-plugin:hello [name]
```

## Arguments

- `name` (optional): The name to greet. Defaults to "World".

## Examples

```
/hello
# Output: Hello, World!

/hello Alice
# Output: Hello, Alice!
```

## Notes

This is an example command demonstrating the command format for Patina plugins.
